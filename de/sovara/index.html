Sovara · Android-Datenschutzwächter

So funktioniert Sovara.

Echtzeit-Erkennung. Sofortige Identifizierung. Warnungen auch bei ausgeschaltetem Bildschirm.

Was passiert, wenn eine App auf Ihr Mikrofon zugreift?

Vier Momente. Vom Zugriff zur Warnung.

01
Erkennung

Eine App schaltet Ihr Mikrofon ein.

Maps. WhatsApp. Ein Spiel. Selbst eine App, die Sie nicht geöffnet haben. Android zeigt einen kleinen Punkt. Das war's.

02
Sovara erkennt es

Sovara erkennt es sofort.

Sovara überwacht das System kontinuierlich. Sobald ein Zugriff beginnt, registriert es Sovara. Latenz: unter 1 Sekunde.

03
Identifizierung

Sovara benennt den Verursacher.

Sowohl die zugreifende App als auch die App im Vordergrund werden identifiziert. Die meisten Konkurrenten zeigen nur, was auf dem Bildschirm ist. Sovara zeigt, wer wirklich den Zugriff angefordert hat.

04
Warnung

Sie werden benachrichtigt — nach Ihren Wünschen.

Ton (8 Optionen + eigener). Vibration (5 Muster). Push-Benachrichtigung. LED-Blitz. Wählen Sie, was zum Moment passt.

So sieht Sovara aus.

Live-Warnung

Sofortige Warnung mit App-Name, Sensortyp und Zeitstempel.

Verlaufsansicht

Chronologisches Protokoll aller Mikrofon- und Kamerazugriffe. Unbegrenzte Dauer.

App-Details

Vollständige Ereignisdetails: App-Name, Paket, Sensor, Dauer, Vordergrund-App.

Whitelist

Apps als vertrauenswürdig markieren. Keine Warnungen mehr von Ihrer Telefon-App.

Wie Sovara 100% lokal bleibt.

Drei technische Fakten, die Sie überprüfen können.

Keine INTERNET-Berechtigung

Sovaras Manifest deklariert keine INTERNET-Berechtigung. Android setzt dies durch. Die App kann physisch keine Netzwerkanfrage stellen — nicht einmal an einen Sovara-Server (der nicht existiert).

Keine Audioaufnahme im Code

Wir nutzen RECORD_AUDIO, um den Mikrofonstatus abzufragen. Das APK enthält jedoch keinen einzigen Aufruf von MediaRecorder.start(), AudioRecord.read() oder einer anderen Audioaufnahme-Methode. Für jeden dekompilierbar und verifizierbar.

Kein Drittanbieter-SDK

Kein Firebase, kein Crashlytics, keine Analytics, kein Werbe-SDK. Die Build-Konfiguration ist offen: Sie können den Abhängigkeitsbaum selbst prüfen.

Für Menschen, denen es wirklich wichtig ist.
